Yet. Another. Web Sudoku.

Sudoku Rules

The objective is to fill the 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids contain all of the digits from 1 to 9. For more information regarding Sudoku, feel free to visit the wikipedia article.

Controls

Besides using a mouse, you can use Arrow Keys, W / A / S / D, or (if you're familiar with VIM-like navigation) H / J / K / L to move the cursor by a single cell. Holding Shift while pressing any direction key will move the cursor over by a box.

If any are errors are present on the grid, pressing E will cause the cursor to cycle through them. Shift + E can be used to move in the opposite direction. Alternatively, X and Shift + X cycle through the errors while clearing the erronous cells of value.

Using Backspace or Delete will clear the current cell of its value. If there is no value it will clear all marked candidates.

Customizing keyboard inputs along with controller support is a planned and upcoming feature.

About This Project

YAWS is a personal project to create a modern and functional web-based sudoku game using standard browser technologies. It is able to solve and generate challenging puzzles in a consistant manner via the use of Web Assembly for its core code. The user interface was created using standard HTML5, CSS, and JavaScript techniques.

Its source code is freely available at the following repositories:

  • User Interface: https://github.com/nxn/yaws
  • Rust puzzle generator and solver: https://github.com/nxn/waffle-iron
